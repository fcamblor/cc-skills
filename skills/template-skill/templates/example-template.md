# Example Template

This is a template file demonstrating the kind of content that belongs in the `templates/` directory.

## Using This Template

Copy this file and customize it for your use case:

```bash
cp templates/example-template.md templates/my-custom-template.md
```

## Template Structure

```
[Boilerplate or starter content]

## Section 1
[Content here]

## Section 2
[Content here]

## Checklist
- [ ] Item 1
- [ ] Item 2
```

## What Goes in Templates

- Code examples (JavaScript, TypeScript, Python, etc.)
- Document templates (markdown, text)
- Configuration templates (YAML, JSON)
- Boilerplate for common patterns
